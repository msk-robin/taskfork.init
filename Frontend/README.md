## Keyboard Shortcuts

- `?` - Show keyboard shortcuts
- `Ctrl+K` / `Cmd+K` - Toggle shortcuts panel
- `Ctrl+Enter` / `Cmd+Enter` - Add/update task
- `Ctrl+Shift+T` / `Cmd+Shift+T` - Cycle through themes
- `Esc` - Close dialogs
